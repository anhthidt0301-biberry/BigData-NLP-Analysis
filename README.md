# Big Data NLP Analysis

This project was developed as part of the Big Data Application and Analysis course.

## Main Features

- Multilingual Word Segmentation
- Stop Word Removal
- Word Frequency Analysis
- Word Cloud
- Sentiment Analysis
- Topic Modeling (LDA)
- BERTopic
- Topic Visualization
- K-means Clustering
- PCA Visualization
- Feature Importance Analysis
- AI-assisted Coding

Python
